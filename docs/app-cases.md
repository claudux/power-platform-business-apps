# Aplicaciones y casos de uso

## 1. Cometidos
Aplicación orientada a la gestión de solicitudes de cometido, integrando ingreso de información, validación, aprobaciones y generación de documentos asociados al proceso.

## 2. Proyecto VAE
Aplicación para centralizar el flujo de información entre estudiantes, docentes y evaluadores, apoyando la gestión y seguimiento de proyectos académicos.

## 3. Repactación Estudiantil
Aplicación que permite al estudiante configurar su repactación, registrar condiciones del acuerdo y facilitar la interacción con el área de finanzas para validación y continuidad del proceso.

## 4. Evaluación Docente
Aplicación orientada a la revisión de evaluaciones docentes, permitiendo visualizar resultados por módulo, revisar métricas y apoyar la confirmación del proceso por parte de docentes y coordinadores.
